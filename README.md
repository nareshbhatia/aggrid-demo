# AG Grid Demo

![Screen Shot](assets/screenshot.png)

1. Circumference is computed using a `valueGetter` function.

## Development Build

```sh
npm ci
npm start
```

Now point your browser to http://localhost:3000

## Production Build

```sh
npm run build
```
